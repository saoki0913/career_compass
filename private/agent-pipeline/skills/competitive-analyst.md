# Competitive Analyst

就活Pass の競合分析・市場ポジショニングの専門スキル。就活支援市場における差別化戦略を立案する。

## 対象ファイル・コンポーネント

- `docs/SPEC.md` — プロダクト仕様
- `docs/marketing/LP.md` — LP戦略・ポジショニング
- `docs/marketing/README.md` — マーケティング概要
- `DESIGN.md` — デザイン指針
- `src/app/(marketing)/pricing/page.tsx` — 料金ページ
- `src/lib/marketing/pricing-plans.ts` — 料金プラン定義

## 分析対象市場

### 就活支援サービスカテゴリ
- ES添削サービス（AI型 / 人力型）
- 就活管理ツール（選考管理、締切管理）
- 就活塾・就活エージェント
- 就活メディア（ONE CAREER、外資就活ドットコム等）
- AI キャリアツール（汎用 AI アシスタントの就活利用含む）

### 主要競合候補
- ES添削カメ — AI ES添削
- 就活SWOT — AI 自己分析
- キャリアチケット — 就活エージェント
- Offer Box — 逆求人型
- ONE CAREER — 就活メディア + ES例文
- ChatGPT / Claude 直接利用 — 汎用 AI による就活支援

## ワークフロー

1. 競合サービスの機能・価格・ターゲットを Web 調査する。
2. 就活Pass との機能比較マトリクスを作成する。
3. 差別化ポイントと弱点を特定する。
4. 戦略提案を作成する。

## 就活Pass の差別化要素

### 強み（仮説）
- ES添削 + 志望動機 + ガクチカ + 企業管理 + 締切管理の統合体験
- AI会話ベースのガクチカ深掘り・志望動機作成
- 企業RAGによる企業固有の添削品質
- ゲストモードで即座に試せる

### ターゲットの明確化
- 就活塾には行かない、自力で進めたい学生
- 超高難度選考向けではなく「迷わず進める管理体験」重視

## 品質基準

- 競合情報は Web 検索で最新データを取得すること。
- 比較は具体的な機能・価格・ターゲットの3軸で行うこと。
- 差別化提案は実行可能なレベルであること。

## 出力

- 日本語で記述。サービス名は原語表記。
- 競合比較マトリクスをテーブル形式で提示。
- SWOT 分析を含む（該当する場合）。
