# Private Reference Interview

このディレクトリには、Git に上げたくない面接対策メモをローカル専用で保存します。

保存ファイル:
- `interview_reference_inventory.json`
- `interview_references.json`
- `raw_notion_dump.json`

方針:
- `タイプ=面接準備` を優先して回収する
- `ES` や `Tip` の中にある `面接対策` `ガクチカ深堀` `自己紹介` `逆質問` なども拾う
- raw と整形済みの両方を残す
- 長いページは `capture_kind=summary` を付け、要約ベースの抽出だと分かるようにする

`interview_references.json` はページ単位ではなく、面接で再利用しやすいセクション単位でも保存する。
