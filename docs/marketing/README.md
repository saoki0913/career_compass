## 0. Public Surface（無料導線）

`/tools` と `/templates` は、LP の主導線を邪魔しない「無料で試せる公開ページ」として扱う。

- 役割は `軽い入口` と `本体への接続` に限定する
- 見た目は LP と同系統の cold trust トーンで揃える
- CTA の主導線は `アプリで続ける`、副導線は各テンプレや文字数確認
- 各ページは単発完結ではなく、保存・添削・締切管理の本体へ戻す

この方針に合わせて、`src/app/tools/*` と `src/app/templates/*` は shared shell で再構成している。

## 0.5 Pricing / Loading UX の現行方針

- `/pricing` は `1 route・1 scroll` で完結する料金ページとして扱う。
- 構成は `hero + billing toggle + plan cards + detailed comparison + FAQ` の一連フローに固定する。
- 内容は `src/lib/marketing/pricing-plans.ts` と `/pricing` 内の比較行を source of truth とし、未実装の特典や比較項目は追加しない。
- loading は `trust surface + skeleton` を標準とし、真っ白な待機面や spinner 単体は使わない。
- 企業一覧・ES一覧の loading では見出しと filter 文脈を残す。

## 1. ビジョン・ポジショニング

**ビジョン**: "就活の不安を、テクノロジーで解決する"

就活塾の1/30の価格でAI就活サポートを提供し、2026年度内に就活生の10%（約5万人）利用を目指す。

**コアメッセージ**: "内定に、最短で近づく就活AI。"

**4つの価値**:
1. **安価**: 就活塾の1/30で質の高いサポート

**差別化要素**: 価格破壊 / ES添削×締切管理×企業研究の3機能統合 / AI特化24時間 / 進捗可視化
