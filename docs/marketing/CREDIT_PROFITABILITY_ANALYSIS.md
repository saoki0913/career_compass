## 6. エージェントチームの総合結論

### 判定: 月額プランは健全、年額プランは要注意

**Standard 月額 (¥3.27/credit):**
- ES添削を除き、全オペレーションで60%以上の粗利率
- 最悪ケース（全クレジットをES添削に集中）でも粗利率5%で黒字維持
- 典型的利用で粗利率51-68%
- **結論: 健全な価格設定** ◎

**Standard 年額 (¥2.72/credit, 16%割引):**
- ES添削集中利用で粗利率-13%の赤字リスク
- ガクチカ・志望動機は粗利率52-61%で安定
- 典型的利用では黒字だが、ES添削ヘビーユーザーで赤字
- **結論: ES添削集中利用のリスク管理が必要** ⚠️

**Pro 月額 (¥3.73/credit):**
- ガクチカ・志望動機は粗利率65-72%で安定
- 企業取得は粗利率81%で優秀
- ES添削1バリアントは粗利率17%で薄利だが黒字
- 最悪ケースでも粗利率17%で黒字維持
- **結論: 安定した価格設定** ◎

**Pro 年額 (¥3.10/credit, 16%割引):**
- ES添削集中利用で粗利率0%（損益分岐点）
- 混合利用では粗利率62%で健全
- **結論: 概ね健全だが薄利に注意** ○

**重要な発見:**
- **年額16%割引により、ES添削のユニットエコノミクスが悪化**
- Standard年額ではES添削集中利用で赤字リスク
- Pro年額ではES添削集中利用で損益分岐点
- ガクチカ・志望動機・企業取得は全プランで健全な粗利率

## 7. 推奨アクション

### 優先度: Critical — 年額プランのES添削赤字対策

#### A. 年額プランの割引率調整

**実施状況**: ⏳ 未実施（検討中、Year 1は16%維持推奨）

**現状の課題:**
- Standard年額 (¥2.72/credit) はES添削集中で-13%の赤字
- Pro年額 (¥3.10/credit) はES添削集中で損益分岐点

**選択肢1: 年額割引率の縮小**
```
現在: 16%割引 (Standard ¥9,800/年, Pro ¥29,800/年)
推奨: 12%割引 (Standard ¥10,400/年, Pro ¥31,600/年)
```
- Standard年額: ¥2.72/credit → ¥2.89/credit（ES添削粗利率 -13% → -2%）
- Pro年額: ¥3.10/credit → ¥3.28/credit（ES添削粗利率 0% → +4%）

**選択肢2: ES添削のクレジット単価調整**
```
ES添削基本クレジット: max(2, ceil(文字数 / 800)) → max(3, ceil(文字数 / 700))
```
- 800字: 2cr → 3cr（収益 ¥8.16年額、コスト ¥10.9、粗利率 -25% → 要検討）
- 1,200字: 2cr → 3cr（同上）

**推奨**: 選択肢1（年額割引率を12%に縮小）が最もシンプルで効果的。

## 11. 前提条件・注意事項

### コスト計算の前提

- トークン数は日本語テキストの推定値（1文字 ≈ 2-3トークン）
- 実際のコストはプロンプト長、RAGコンテキスト量、リトライ回数により変動
- ES添削のクレジットコスト計算式: `max(2, ceil(charCount / 800))`、上限5 (`src/lib/credits/cost.ts`)
- 為替レート: ¥1 ≈ $0.0065 (2026年2月時点の概算)
- Cross-encoderリランキング (`hotchpotch/japanese-reranker-small-v2`) はローカル実行のためAPIコスト不要
- LLMリランキング (`MODEL_RAG_RERANK`) はコード上未使用（デッドコード）

### プラン設計の前提

- **クレジット数はコード上300/800（PricingSection.tsx, Stripe config）を使用**
- 旧ドキュメントの100クレジット記載は誤り（本ドキュメントで修正済み）
- **3x ES添削機能は実際には使用されていない**（分析から除外）
- Free/Guestプランは収益を生まないため、有料転換率が全体収益に大きく影響
- 年額プラン選択率は初年度30%想定、Year 2以降増加見込み

### 分析の制限事項

- インフラコスト（Supabase, Vercel, Railway等）は本分析に含まない
- カスタマーサポートコスト、マーケティングコストは含まない
- 季節性（3-4月ピーク、8-9月閑散期）は考慮していない
- チャーン率5%/月は保守的推定（実際は就活終了時期に集中）
