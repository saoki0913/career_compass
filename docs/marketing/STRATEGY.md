## 1. ビジョン・ポジショニング

**ビジョン**: "就活の不安を、テクノロジーで解決する"

就活塾の1/30の価格でAI就活サポートを提供し、2026年度内に就活生の10%（約5万人）利用を目指す。

**コアメッセージ**: "内定に、最短で近づく就活AI。"

**4つの価値**:
1. **安価**: 就活塾の1/30で質の高いサポート
2. **迷わない**: AIが今日やるべきタスクを提示
3. **締切を落とさない**: 自動抽出・通知で機会損失ゼロ
4. **ES品質向上**: AI添削で通過率を上げる

**差別化要素**: 価格破壊 / ES添削×締切管理×企業研究の3機能統合 / AI特化24時間 / 進捗可視化

## 3. ターゲットペルソナ（5類型）

### ペルソナ1: 「不安先行型」佐藤 美咲
- **属性**: 21歳、中堅私大文学部3年、広告・サービス志望、東京
- **ペイン**: 情報過多で優先順位不明 / ES添削相手なし / 締切漏れ / 企業研究が浅い
- **Aha Moment**: 「今日やることが分かりやすい！」（ダッシュボード初回表示）
- **心理トリガー**: Loss Aversion「締切を落とさない」/ Anchoring「就活塾月3-10万 vs 月980円」
- **チャネル**: Instagram、X、友人口コミ

### ペルソナ2: 「効率追求型」田中 翔太
- **属性**: 21歳、国立大工学部3年、IT・メーカー・コンサル志望、大阪
- **ペイン**: 非効率な作業が嫌 / ESは書けるがフィードバック不足 / ツール分散
- **Aha Moment**: AI添削の具体的改善提案を見た瞬間
- **心理トリガー**: Authority「AI添削エンジンの分析力」/ 効率性訴求
- **チャネル**: Google検索、Qiita、note、YouTube

### ペルソナ3: 「地方ハンデ型」鈴木 あかり
- **属性**: 20歳、地方国立大経済学部3年、金融・公務員志望、地方在住
- **ペイン**: OB訪問困難 / 情報格差 / 就活塾が地元にない
- **Aha Moment**: 「地方にいても都市部と同じサポート」を実感
- **心理トリガー**: Scarcity「今月の無料枠あと○回」/ 地方×AI活用の新しさ
- **チャネル**: Instagram、LINE、友人口コミ

### ペルソナ4: 「直前追い込み型」高橋 大樹
- **属性**: 22歳、私大法学部4年、商社・金融志望、東京
- **ペイン**: 出遅れて焦り / 大量ESを短期間で仕上げ必要 / 締切ラッシュ
- **Aha Moment**: 「今からでも間に合う」→ 即日ES添削完了
- **心理トリガー**: Urgency「今からでも間に合う」/ Social Proof「○○人利用中」
- **チャネル**: X、YouTube、友人口コミ

### ペルソナ5: 「コスパ重視型」山本 結衣
- **属性**: 21歳、私大商学部3年、メーカー・IT志望、東京
- **ペイン**: 就活塾は高額 / 無料ツールの品質不安 / 費用対効果を重視
- **Aha Moment**: 「月980円で就活塾レベルの添削」を実感
- **心理トリガー**: Anchoring「1/30の価格」/ Reciprocity「無料30クレジット」
- **チャネル**: Google検索、比較サイト、口コミ

## 4. 競合分析

### 4.1 直接競合

| サービス | 主要機能 | 料金 | 脅威度 |
|---------|---------|------|--------|
| SmartES | ES生成・添削 | 5回無料→有料 | 高 |
| 内定くんAI | ES添削（LINE） | 基本無料 | 中 |
| 就活AI byジェイック | ES添削・面接 | 無料 | 中 |
| ChatGPT/Claude | 汎用AI | ¥0-20/月 | 中（就活特化でない） |

### 4.2 コア機能比較

| 機能 | 就活Pass | SmartES | 内定くんAI | ChatGPT | 就活塾 |
|------|---------|---------|----------|---------|--------|
| ES添削（AI） | 8種スタイル | 1種 | 企業別 | 汎用 | 人手 |
| 締切管理 | 自動抽出 | - | - | - | 手動 |
| 企業研究（RAG） | AI検索 | - | - | 汎用 | 対応 |
| 月額料金 | ¥0-980 | 有料 | ¥0 | ¥0 | ¥30,000+ |

### 4.3 技術的差別化

| 技術 | 就活Pass | 競合 |
|------|---------|------|
| RAG（ChromaDB + BM25 + Rerank） | 独自実装 | ほぼなし |
