## コンテンツ戦略概要

### 戦略目標

**主目標**: オーガニック流入とSNSバイラルで月間5,000人の新規ユーザー獲得（content CAC: ¥0-500）

**副目標**:
- SEO流入: 月間20,000PV達成（6ヶ月以内）
- SNSフォロワー: Twitter 5,000人、TikTok 10,000人、Instagram 3,000人（3ヶ月以内）
- LINE登録: 月間500人の新規登録
- コンテンツ経由のコンバージョン率: 3-5%

### 4つのコンテンツピラー

#### Pillar 1: ES作成のテクニック
**目的**: SEO流入 + 有料プラン転換

**コンテンツタイプ**:
- ESの書き方ガイド（基本〜応用）
- 業界別・職種別ES例文集
- ES添削のビフォーアフター事例
- AI添削の活用法

**ターゲットキーワード**:
- ES 書き方（検索ボリューム: 18,000/月）
- エントリーシート 添削（4,500/月）
- ガクチカ 例文（12,000/月）
- 志望動機 書き方（8,100/月）

**ファネル連携**: Awareness → Consideration（添削機能の無料トライアルへ誘導）

---

#### Pillar 2: 締切・スケジュール管理
**目的**: 差別化機能のアピール + LINE登録

**コンテンツタイプ**:
- 就活スケジュールカレンダー
- 締切管理のコツ
- 時期別やるべきことチェックリスト
- 企業別締切一覧（随時更新）

**ターゲットキーワード**:
- 就活 スケジュール（9,900/月）
- インターン 締切（3,600/月）
- 本選考 スケジュール（2,400/月）

**ファネル連携**: Awareness → Retention（LINE登録で締切通知受け取りへ誘導）

---

#### Pillar 3: 企業研究ノウハウ

## 週次コンテンツテンプレート

創業者1人で運営可能な週次スケジュール。

### 月曜日: ブログ記事公開日

**タスク**:
1. 新規ブログ記事を公開（2,000-3,000文字）
2. SEO最適化チェック（タイトル、メタディスクリプション、見出し、内リンク）
3. SNSで告知投稿（Twitter、Instagram、LINE）
4. 関連する過去記事への内部リンク追加

**時間配分**: 3-4時間

**テンプレート**:
```markdown
## [記事タイトル]（ターゲットキーワード含む）

### 導入（15%）
- 読者の悩みを明確化
- 記事で得られるベネフィット提示

### 本文（70%）
- H2見出し3-5個
- 具体例・事例・データ
- 画像・図表2-3点

### まとめ（15%）
- 要点の再確認
- CTA（無料登録 or 添削機能を試す）

### 内部リンク
- 関連記事2-3本へのリンク
```

**公開チェックリスト**:
- [ ] タイトルにキーワード含む（50文字以内）
- [ ] メタディスクリプション設定（120文字）
- [ ] H2/H3見出し構造が適切
- [ ] 画像にalt属性設定
- [ ] 内部リンク2本以上
- [ ] CTA明確
- [ ] モバイル表示確認
