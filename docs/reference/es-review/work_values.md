# `work_values`：働くうえで大切にしたい価値観の構成ヒント

このファイルは、品質フィルタ後の参考ESから抽出した構成パターンをもとに、文字数制限が変わっても高品質なESを書けるように整理したものです。

使い方はシンプルです。まず設問タイプを確認し、次に指定文字数帯の型を選び、最後に自分の経験・企業固有要素・成果数字を差し込んでください。

文字数帯は次の6段階に対応しています。

- `〜100字`
- `100〜200字`
- `200〜300字`
- `300〜400字`
- `400〜500字`
- `500字以上`


## 1. この設問で評価されること

`work_values` では、スキルではなく「どのような姿勢で働く人か」が見られます。価値観は抽象的になりやすいため、経験に基づいて語ることが重要です。

評価されやすい回答は、次の条件を満たします。

- 価値観が一言で明確である
- 価値観が実体験に基づいている
- 経験から得た学びが仕事観へ抽象化されている
- 志望企業・職種でその価値観をどう体現するかがある
- 「大切にしたい」だけでなく、具体行動に落ちている

## 2. 基本の論理構成

```text
① 大切にしている価値観を一言で示す
② その価値観が形成された経験を述べる
③ 経験から得た学びを抽象化する
④ 志望企業・職種でその価値観をどう体現するか
⑤ 将来の貢献につなげる
```

強い流れは次の通りです。

```text
価値観
→ 形成された経験
→ そこから得た学び
→ 仕事での行動
→ 貢献
```

## 3. 価値観を強くする言い換え

| 弱い価値観 | 強い価値観 |
|---|---|
| 挑戦を大切にしたい | 自ら機会を創り、周囲を巻き込みながら未知の課題に挑戦する |
| 協調性を大切にしたい | 多様な個性を掛け合わせ、共通目標に向けてチームの力を最大化する |
| お客様第一 | 相手の立場や潜在課題を理解し、期待を超える価値提供を目指す |
| 成長したい | 挑戦から得た学びを次の行動に還元し、組織成果につなげる |
| 誠実さ | 相手や成果に対して責任を持ち、困難な状況でも信頼を積み上げる |

## 4. 文字数帯別の設計

### 4-1. `〜100字`

価値観、根拠、仕事での体現を一文でまとめます。経験の詳細は入れず、価値観の輪郭を示します。

| 要素 | 配分目安 | 書くこと |
|---|---:|---|
| 価値観 | 35% | 何を大切にするか |
| 根拠 | 30% | 経験から得た学び |
| 体現 | 35% | 仕事でどう行動するか |

**テンプレート**

```text
私が大切にしたい価値観は〇〇である。△△の経験から□□の重要性を学んだ。仕事でもAを意識し、周囲と協働してBに貢献したい。
```

### 4-2. `100〜200字`

価値観が形成された経験を一文入れます。価値観と企業での行動を必ずつなげます。

| 要素 | 配分目安 | 書くこと |
|---|---:|---|
| 価値観 | 20% | 一言で明示 |
| 経験 | 30% | 形成された背景 |
| 学び | 25% | 経験から得た仕事観 |
| 体現 | 25% | 志望先での行動 |

**テンプレート**

```text
私が大切にしている価値観は〇〇である。△△の経験では、□□という状況に対しAを行い、Bという成果を得た。この経験から、Cの重要性を学んだ。貴社ではDの場面でこの価値観を体現し、Eに貢献したい。
```

### 4-3. `200〜300字`

経験と学びをバランスよく書けます。経験の説明が長くなりすぎると `gakuchika` になるため、価値観に戻す文を入れます。

| 要素 | 配分目安 | 書くこと |
|---|---:|---|
| 価値観 | 15% | 大切にする姿勢 |
| 経験 | 30% | 価値観が形成された場面 |
| 学び | 25% | 仕事観への抽象化 |
| 企業接続 | 20% | その価値観が活きる業務 |
| 貢献 | 10% | どう体現するか |

**テンプレート**

```text
私は「〇〇」を働くうえで大切にしたい。この価値観は、△△の経験から形成された。当時、□□という課題があったが、私はAを行い、周囲と協働してBを実現した。この経験から、個人の力だけでなく、多様な人を巻き込み、共通目標に向かうことで大きな成果を生み出せると学んだ。貴社のCという仕事では、Dとの調整やEの視点が不可欠である。私はこの価値観を活かし、Fに貢献したい。
```

### 4-4. `300〜400字`

価値観の形成過程と、企業での体現方法を具体化できます。「なぜその価値観を大切にするのか」を深く説明します。

| 要素 | 配分目安 | 書くこと |
|---|---:|---|
| 価値観 | 10% | 一言で示す |
| 原体験 | 30% | 価値観が形成された経験 |
| 学び | 25% | 抽象化された仕事観 |
| 企業接続 | 20% | その価値観が重要な理由 |
| 体現 | 15% | 入社後の行動 |

**テンプレート**

```text
私が働くうえで大切にしたい価値観は、〇〇である。この価値観は△△の経験から形成された。当時、□□という状況で、私はAを実行した。最初はBという困難があったが、Cを意識して周囲と向き合うことで、Dという成果につながった。この経験から、成果を出すには個人の能力だけでなく、相手の立場を理解し、信頼関係を築きながら行動することが重要だと学んだ。貴社のEという仕事では、多様な関係者と協働しながら価値を届ける必要があると考える。私はこの価値観を体現し、Fに貢献したい。
```

### 4-5. `400〜500字`

価値観、原体験、仕事観、企業接続、将来の姿勢を丁寧に書けます。複数の価値観を並べず、一つの価値観を深掘りします。

| 要素 | 配分目安 | 書くこと |
|---|---:|---|
| 価値観 | 10% | 大切にしたいこと |
| 原体験 | 25% | 形成された経験 |
| 行動・成果 | 20% | その価値観が表れた行動 |
| 学び | 20% | 仕事観への抽象化 |
| 企業接続 | 15% | 企業・職種での重要性 |
| 体現 | 10% | どう行動するか |

**テンプレート**

```text
私が働くうえで大切にしたい価値観は、〇〇である。これは、△△の経験を通じて形成された。当時、□□という課題があり、チーム内でもAという状況が生まれていた。私は、成果を出すにはBが必要だと考え、Cを実行した。特に、Dを意識することで、メンバーが主体的に行動できる状態を作った。その結果、Eという成果につながった。この経験から、仕事においては単に自分の役割を果たすだけでなく、相手の立場を理解し、周囲と共通の目的に向かって行動することが重要だと学んだ。貴社のFという業務では、多様な関係者と協働しながら価値を生み出す必要があると考えている。私はこの価値観を体現し、信頼を積み上げながらGに貢献したい。
```

### 4-6. `500字以上`

500字以上では、価値観の形成過程と、企業での体現方法を段落で整理します。経験を二つ入れる場合は、どちらも同じ価値観を補強するものにします。

**推奨構成**

```text
第1段落：大切にしたい価値観
第2段落：価値観が形成された経験
第3段落：経験中の行動と成果
第4段落：経験から得た仕事観
第5段落：企業・職種でその価値観が重要な理由
第6段落：入社後の体現方法
```

**テンプレート**

```text
私が働くうえで大切にしたい価値観は、〇〇である。私にとって〇〇とは、単にAを意味するのではなく、Bの状況でもCを意識し、成果に向けて行動する姿勢である。

この価値観は、△△の経験から形成された。当時、□□という課題があり、チーム全体の成果に影響していた。私は、問題の背景にはDとEがあると考えた。

そこで、第一にFを実行し、関係者の認識をそろえた。第二にGを行い、メンバーが主体的に行動できる状態を作った。実行時にはHという困難もあったが、Iを意識して対話を重ねた結果、Jという成果につながった。

この経験から、仕事で成果を出すには、個人の能力だけでなく、相手の立場を理解し、信頼関係を築きながら共通目標に向かう姿勢が重要だと学んだ。

貴社のKという仕事では、多様な関係者と協働し、顧客や社会に価値を届けることが求められると考えている。そのため、私が大切にする〇〇という価値観は、貴社で働くうえでも重要だと考える。

入社後は、この価値観を日々の行動で体現し、周囲から信頼される姿勢を積み上げながら、Lに貢献したい。
```

## 5. 文字数を増やすときに足す順番

1. 価値観の定義を具体化する
2. 価値観が形成された経験を説明する
3. 経験から得た学びを仕事観へ抽象化する
4. 企業・職種でその価値観が重要な理由を追加する
5. 入社後の体現方法を具体化する

## 6. 文字数を削るときに残す順番

1. 価値観
2. 経験の根拠
3. 学び
4. 仕事での体現
5. 詳細な背景説明

## 7. NGパターン

```text
私は挑戦と成長を大切にしたい。社会人になっても多くのことを学び、成長していきたい。
```

弱い理由は、価値観の根拠と仕事での行動イメージがないためです。

## 8. 最終チェック

- 価値観が一言で明確か
- 価値観が実体験に基づいているか
- 経験の説明が長すぎず、価値観の説明に従属しているか
- 企業や職種でその価値観がどう活きるか書かれているか
- 「大切にしたい」だけでなく、どう行動するかが明確か
