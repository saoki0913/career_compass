# Product Context - 就活Compass (シューパス)

## Overview
就活支援アプリ「就活Pass」- 就活塾に行かない不安層に対して、AIと進捗管理で「安価に、迷わず、締切を落とさず、ESの品質を上げる」ことを実現する。

## Target Users
- 情報弱者寄り、就活塾回避層
- 外資投資銀行/戦略コンサル/総合商社など超高難度層は主ターゲットではない

## Core Value Proposition
1. **安価**: 就活塾より低コストで質の高いサポート
2. **迷わない**: AIが今日やるべき1タスクを提示
3. **締切を落とさない**: 自動抽出・通知・カレンダー連携
4. **ESの品質向上**: AI添削とガクチカ深掘り

## MVP Features (Phase 1)
- ES自動添削（スコア/Top3/リライト）
- ガクチカ深掘りBot
- 進捗管理（企業登録→公式情報抽出→締切承認→通知→タスク提案）
- ES作成ドキュメント（保存・企業/応募枠紐づけ）
- ESテンプレ共有/ギャラリー
- 企業登録（公式採用ページ候補提示）
- カレンダー連携（Google/アプリ内）
- クレジットシステム

## Excluded from MVP
- 企業別攻略カード
- コミュニティ機能

## Pricing
- **Guest**: 無料（制限あり）
- **Free**: 無料（ログイン後）
- **Standard**: 980円/月
- **Pro**: 2,980円/月

## Key Metrics
- クレジット消費（成功時のみ）
- 企業情報取得/更新の無料回数
- ES添削の文字数ベース消費
